## 📂 Project Experience

### 🎯 KananRPC - A Lightweight Event-Driven RPC Framework Based on Vert.x
**Project Duration**: 2025.9 - 2026.1

#### Project Overview
This framework is built with Java, integrating Etcd and Vert.x technologies with custom protocol extensions. It enables quick integration through Spring Boot Starter dependency, supporting annotation-based configuration. The framework employs SPI mechanism for flexible dynamic feature extension, implementing core functionalities including load balancing strategies, retry logic, and fault tolerance mechanisms.

#### Core Technical Implementation
- 🔧 **Message Queue & Processing**: Handle high-concurrency calls, serialize/deserialize data, manage network services, and declare service relationships
- 🌐 **HTTP Service**: Build HTTP services based on Vert.x, supporting asynchronous and bidirectional communication with high performance
- 🔐 **Concurrency Safety**: Utilize thread-safe ConcurrentHashMap for local service mapping, support name-based service lookup and concurrent deserialization
- 📦 **Dynamic Serialization**: Define generic interfaces for extensibility, implement JdkSerializer based on Java reflection and ByteArray streams for network object transmission
- ⚡ **Request Processing**: Handle requests asynchronously using Vert.x Handler, perform deserialization, determine service implementation, and invoke via reflection
- 🔄 **HTTP Proxy**: Generate factory-style HTTP proxies for service interfaces using JDK dynamic proxy, enabling seamless remote invocation
