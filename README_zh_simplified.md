## 📂 项目经验

### 🎯 KananRPC - 基于Vert.x的轻量级事件驱动型RPC框架
**项目周期**: 2025.9 - 2026.1

#### 项目概述
该框架基于Java构建，整合Etcd与Vert.x技术，实现自定义协议扩展。可通过Spring Boot Starter依赖快速集成，支持注解式配置。框架采用SPI机制实现灵活的动态功能扩展，完成负载均衡策略、重试逻辑、容错机制等核心功能。

#### 核心技术实现
- 🔧 **消息队列 & 处理**: 处理高并发调用、数据序列化/反序列化、网络服务管理、服务关系声明
- 🌐 **HTTP服务**: 基于Vert.x构建HTTP服务，支持异步双向通信，具备高性能特性
- 🔐 **并发安全**: 采用线程安全的ConcurrentHashMap实现本地服务映射，支持基于名称的服务查找与并发反序列化
- 📦 **动态序列化**: 定义泛型接口实现扩展性，基于Java反射与ByteArray流实现JdkSerializer，完成网络对象传输
- ⚡ **请求处理**: 采用Vert.x Handler异步处理请求，执行反序列化、判断服务实现、通过反射完成调用
- 🔄 **HTTP代理**: 利用JDK动态代理为服务接口生成工厂式HTTP代理，实现无缝远程调用
