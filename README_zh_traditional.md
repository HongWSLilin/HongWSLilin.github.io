## 📂 專案經歷

### 🎯 KananRPC - 基於Vert.x的輕量級事件驅動型RPC框架
**專案週期**: 2025.9 - 2026.1

#### 專案概述
該框架基於Java建構，整合Etcd與Vert.x技術，實現自訂協定擴充。可透過Spring Boot Starter依賴快速整合，支援註解式設定。框架採用SPI機制實現靈活的動態功能擴充，完成負載平衡策略、重試邏輯、容錯機制等核心功能。

#### 核心技術實現
- 🔧 **訊息佇列 & 處理**: 處理高併發呼叫、資料序列化/反序列化、網路服務管理、服務關係宣告
- 🌐 **HTTP服務**: 基於Vert.x建構HTTP服務，支援非同步雙向通訊，具備高效能特性
- 🔐 **併發安全**: 採用執行緒安全的ConcurrentHashMap實現本機服務對應，支援基於名稱的服務查詢與併發反序列化
- 📦 **動態序列化**: 定義泛型介面實現擴充性，基於Java反射與ByteArray串流實現JdkSerializer，完成網路物件傳輸
- ⚡ **請求處理**: 採用Vert.x Handler非同步處理請求，執行反序列化、判斷服務實現、透過反射完成呼叫
- 🔄 **HTTP代理**: 利用JDK動態代理為服務介面生成工廠式HTTP代理，實現無縫遠端呼叫
